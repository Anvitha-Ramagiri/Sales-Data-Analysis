# Sales Data Analysis

## Overview

This project analyzes sales data to uncover business insights, identify sales trends, evaluate product performance, and support data-driven decision-making. The project uses Python in Visual Studio Code (VS Code) for data analysis and Matplotlib for visualizations. An interactive Power BI dashboard is created to present key findings and business metrics.

---

## Objectives

* Analyze overall sales performance.
* Identify top-selling products and categories.
* Examine monthly and yearly sales trends.
* Compare regional sales performance.
* Create visualizations and dashboards for business reporting.
* Generate actionable business insights.

---

## Dataset

The dataset contains sales transaction records, including:

* Order ID
* Order Date
* Product Name
* Category
* Sales Amount
* Quantity Sold
* Customer Information
* Region
* Profit

## How to Run

### Clone the Repository

```bash
git clone https://github.com/your-username/sales-data-analysis.git
```

### Install Dependencies

```bash
pip install pandas numpy matplotlib
```

### Run the Project in VS Code

1. Open the project folder in VS Code.
2. Open the terminal.
3. Install the required libraries.
4. Run the Python script:

```bash
python sales_analysis.py
```

5. View generated charts and insights.
6. Open the Power BI dashboard (`.pbix`) file to explore interactive reports.
